There might be a few Characters that when Executed in the Program may make the Program act Differently and hence Close our Connection, hence in those Cases, we actually want to Delete/ Remove those Bad Characters.

- /x00 is a Default Bad Character
- /B0 is NOT a Bad Character
- If 2 Consecutive Bad Character are there, then only the First Charcter is Bad, the Second One is Okay.

We have to look closely through the Hex Dump and if see something unusual, then it might be a bad character.

https://github.com/cytopia/badchars
